pso.evac
========

Source code used in "PSO-Based Evacuation Simulation Framework"

----------

Usage
-----

#### Get a brief description

> help Evacuation;

#### Simulate scene 1 and save data to **data/demo.mat** without display

> Evacuation(1 , 'demo' , 0);
